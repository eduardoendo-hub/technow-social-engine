# Tech Now Marketing Site — UI Kit

A high-fidelity recreation of a marketing/landing surface for Tech Now, the
AI-automation business. Dark Tiffany aesthetic — Deep Teal background, Tiffany
brand color, Rose Blush accent dot.

## Files

- `index.html` — full page composing the components below
- `Header.jsx` — top nav with Tech Now logo
- `Hero.jsx` — headline + sub + dual CTA
- `LogoBar.jsx` — "trusted by" placeholder rail
- `FeatureGrid.jsx` — 3-up feature cards
- `WorkflowDemo.jsx` — terminal/workflow card with status pills
- `CTABanner.jsx` — full-width tiffany CTA
- `Footer.jsx` — minimal footer

## Notes

This is a recreation, not new design. Visuals are taken from the brand sheet
(`uploads/brand_sheet.html`) and palette (`uploads/palette_tiffany.html`).
Tech Now copy is in PT-BR per the company description; the kit mirrors that
voice ("Automação, agora.").

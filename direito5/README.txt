Direito 5.0 — Landing Page
==========================

Conteúdo do pacote:
  index.html                       → página principal (HTML estático)
  styles/direito5-lp.css           → estilos
  js/direito5-lp.js                → animações / interações
  assets/                          → imagens (logos, palestrantes, hero)

Como subir no servidor:
-----------------------
1. Suba a pasta inteira (preservando a estrutura) na raiz do site
   ou em um subdiretório (ex: /direito-5-0/).
2. Aponte o domínio/rota para o index.html.
3. Não há build, não há dependências de servidor — basta servir como
   arquivos estáticos (Apache, Nginx, S3, Vercel, Netlify, etc.).

Fontes:
  Carregadas direto do Google Fonts (Ubuntu, Cormorant Garamond, Ubuntu Mono)
  via <link> no <head>. Requer acesso à internet do navegador do visitante.
